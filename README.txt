Le script collecte.sh permet d'executer tous les algorithmes sur tous les exemplaires. Aucun argument requis.
Il utilise le script tp.sh qui fonctionne tel que décrit dans l'énoncé.

Les scripts traceGraph*.sh permettent de tracer les graphs necessaires aux differents test, aucun argument requis.

Le repertoire src contient les algorithmes ainsi que quelques scripts utilitaires qui nous ont permis de mener à bien ce laboratoire. Ces-derniers ne prennent pas d'arguments.

Tous les graphs se trouvent dans le dossier graphs

Les temps d'execution sont dans le dossier results : raw contient les temps d'executions générés par le script collecte.sh et means les temps d'execution moyens. Le reste sont les différents calculs effectués pour les tests.
